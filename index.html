RoboMaze

Your task is to lead your robot across the maze (without leaving the safety of the path!). You start at position (1,1) and you need to reach coordinate (98,98).

Go to the live map

You can control your robot using the RoboMaze REST API endpoint at https://robomaze.skadge.org/. See below for the available requests.

A robot that remains inactive for more that 10min is automatically removed.

You need to wait at least 200ms between two commands. Otherwise, your command will be ignored.

Move a robot

To move your robot in the map, use the action move, with two options: the name of your robot, and the direction of the move: North N, South S, East E or West W.

If your robot does not exist yet (or has been killed), calling move will first create it at the coordinate (1,1)

Returns: a JSON object 

[<move successful?>, [<obstacle at N?>, <S?>, <E?>, <W?>]]
Examples: 
# from a terminal
curl 'https://robomaze.skadge.org/move/Wall-E/E'
# in python
import json
import requests

REST_ENDPOINT = "https://robomaze.skadge.org/"
requests.get(REST_ENDPOINT + "move/Wall-E/N")

Remaining life of a robot

Use the action life, with the name of your robot (as a JSON string).

Returns: the remaining life of the robot, or 0 if the robot does not exist (or has died). Examples: 

# from a terminal
curl 'https://robomaze.skadge.org/life/Wall-E'
# in python
import json
import requests

REST_ENDPOINT = "https://robomaze.skadge.org/"
requests.get(REST_ENDPOINT + "life/Wall-E")
